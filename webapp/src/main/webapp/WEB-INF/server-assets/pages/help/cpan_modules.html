CPAN modules

A number of CPAN modules are used in the processing of MAGE-TAB files.

Bio::MAGETAB

The Bio::MAGETAB module was written by Tim F. Rayner and provides the core set of classes used to support the perl MAGE-TAB API. This module provides a set of data structures and type constraints which help to reliably handle data in MAGE-TAB format. More information on this module can be found here

EBI::FGPT::FuzzyRecogniser

The EBI::FGPT::FuzzyRecogniser module is used for ontology mapping. It takes as input an ontology file (OWL/OBO/OMIM/MeSH) and a term. The module contains the find_match method which finds the best match for the supplied term in the given ontology. More information on this module can be found here
